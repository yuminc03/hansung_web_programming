About This Website 👩🏻‍💻

대학생이 되어 웹 프로그래밍 수업을 듣고 만들게 된 웹사이트입니다.
고등학생 때도 웹 프로그래밍을 배웠는데 웹 개발자가 아닌 앱 개발자로 일하면서 웹 개발을 잊었지만 다시 배우면서 잊고 있던 기술에 대한 기억을 되살리고 처음 배웠을 때의 그 마음가짐을 다시 가졌습니다.

웹 개발을 처음부터 다시 배워서 좋았던 점은 고등학생 시절에 진도를 나가느라 정확히 모르고 넘어갔던 부분들을 다시 배울 수 있었습니다! 😄
